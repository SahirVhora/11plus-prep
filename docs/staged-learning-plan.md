# Staged Learning Plan

This plan gives parents and pupils a safer route through the 11+ content without changing the quiz engine.

## Stage 1: Diagnostic Baseline

- Run one short quiz per subject: English, Maths, Verbal Reasoning, Non-Verbal Reasoning.
- Record score, time pressure, and confidence.
- Use weak-area analysis to pick the first two focus topics.

## Stage 2: Skill Building

- Practise topic drills before mixed tests.
- Keep sessions short: 15-20 minutes for Year 5, slightly longer near exam season.
- Retake only the missed topic after reviewing explanations.

## Stage 3: Regional Alignment

- Select the target region and check the regional information panel before starting timed practice.
- Prioritise question styles used by the selected board.
- Re-run `npm run validate-regions` after changing region metadata or question banks.

## Stage 4: Timed Exam Conditioning

- Move to timed mixed sessions only after topic accuracy is consistently above 75%.
- Review wrong answers immediately after each session.
- Track recurring mistakes by subject and question type.

## Stage 5: Final Readiness

- Use full mixed practice in the final phase.
- Stop adding new topics close to the exam; focus on accuracy, pacing, and confidence.
- Export or screenshot result summaries for parent review.
